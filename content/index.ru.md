## Введение
Бот Twitch - это настраиваемый бот на основе ChatGPT, предназначенный для предоставления различных функций для стримеров Twitch и их зрителей. В этой документации представлен обзор доступных команд в Twitch Bot и как их использовать.

## Команды

### !yg help
Синтаксис: `!yg help`

Описание: Команда `!yg help` отображает список доступных команд и их описаний, чтобы помочь пользователям понять, как взаимодействовать с Twitch Bot.

Пример использования:
```
!yg help
```

### !yg bio
Синтаксис: `!yg bio`

Описание: Команда `!yg bio` отображает биографию Twitch Bot, которую можно настроить пользователем.

### !yg bio <text>
Синтаксис: `!yg bio <text>`

Описание: Команда `!yg bio` позволяет пользователям установить пользовательскую биографию для Twitch Bot, давая ей личность и предоставляя информацию, такую ​​как место работы, обычную деятельность и ее любимые стримеры.

Пример использования:
```
!yg bio "Привет, я ChatGPT, ваш дружелюбный Twitch-бот! Я работаю в Twitch HQ, люблю взаимодействовать с зрителями и наслаждаюсь просмотром трансляций со своим любимым стримером, @TwitchStreamer!"
```

### !yg trigger_words
Синтаксис: `!yg trigger_words`

Описание: Команда `!yg trigger_words` отображает текущие ключевые слова или фразы, которые, когда они упоминаются в чате, активируют Twitch-бота и вызывают ответ.

### !yg trigger_words <слова, разделенные запятой>
Синтаксис: `!yg trigger_words <слова, разделенные запятой>`

Описание: Команда `!yg trigger_words` позволяет пользователям установить ключевые слова или фразы, которые, когда они упоминаются в чате, активируют Twitch-бота и вызывают ответ. Это позволяет пользователям настроить ключевые слова, которые активируют бота и взаимодействуют с аудиторией.

Пример использования:
```
!yg trigger_words привет, приветствую, приветствия, бот
```

Примечание: Ключевые слова должны быть разделены запятыми без пробелов.

### !yg welcome_message
Синтаксис: `!yg welcome_message `
Описание: Команда `!yg welcome_message` отображает текущее сообщение приветствия, которое будет отображаться при присоединении нового зрителя к чату.

### !yg welcome_message (true|false)
Синтаксис: `!yg welcome_message (true|false)`

Описание: Команда `!yg welcome_message (true|false)` позволяет пользователям включать или отключать сообщение приветствия, которое будет отображаться при присоединении нового зрителя к чату.

Пример использования:
```
!yg welcome_message true
```
